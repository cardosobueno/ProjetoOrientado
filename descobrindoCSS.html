Descobrindo CSS

border-radius

A primeira propriedade CSS que procurei foi a border-radius . Ela pode ser usada para suavizar a borda de imagens, caixas de texto, etc.

Abaixo vemos uma imagem sem borda alguma, e do lado direito a mesma imagem com 50 pixels de borda (border-radius: 50px)

Fonte customizada

Para usar uma fonte baixada da internet, é muito simples:

Basta baixar uma fonte .otf ou .ttf; colocar o arquivo numa pasta com o nome "fonts" onde essa deve ser encontrada dentro da pasta "css" junto com o arquivo "main.css"

Após isso, vá no arquivo main.css e crie a regra @font-face. Dentro desta regra, insira o local que o arquivo da fonte se encontra e insira o formato do arquivo.

Exemplo:

@font-face { font-family: "FonteCustom"; src: url('fonts/JosefinSans-Regular.ttf') format('truetype'); }

Na hora de utilizar a fonte, não se esqueça de inclui-lá na propriedade font-family.

Centralizando no centro itens centrais

Tive um pouco de dificuldade para centralizar caixas de texto e imagens. Felizmente o site oficial do W3C tem uma página ensinando.

Utilizei da propriedade "margin-left: auto;" e "margin-right: auto;" e os itens ficaram centralizados perfeitamente.

Animação simples de transição

Usando a propriedade "transition", é possível criar uma pequena animação em um elemento. Utilizei isso no H1 de todas as páginas, causando uma pequena animação ao abri-lá.

Exemplo do código que usei:

h1{ background-color: #14213d; color:white; padding: 10px; border-radius: 15px; transition: 0.75s; }

Existem também funções como ease-in-out que suavizam ainda mais a animação de transição.

Sombra em elementos.

Uma boa maneira de enfeitar elementos de uma página é utilizando a propriedade "box-shadow". Essa propriedade projeta uma sombra atrás do elemento. O usuário também pode manipular os valores para mudar a posição, cor, e força da sombra.

Exemplo do código que usei:

box-shadow: 0 0 30px rgba(0, 0, 0, 0.15);

Uma ferramenta excelente para criação de sombras pode ser encontrada nesse link.

Com ela você monta sombras mexendo numa interface criada pela Mozilla Developer Network.

Linhas com cores diferentes

Nas tabelas de sites favoritos e curiosidades, achei interessante incluir linhas com cores diferentes em cada item para facilitar a leitura dos dados.

Usando a pseudoclasse "nth-of-type" em um elemento de tabela, você consegue colocar cada linha de uma cor distinta.

Exemplo do código que usei:

.tabelaCuriosa tbody tr:nth-of-type(even) { background-color: #e5e5e5; color: black; }

.tabelaSites li:nth-of-type(even) { background-color: #e5e5e5; }
